Usable Burger

View Code

I was assigned this project while interviewing with Usable Life for an intern C#/Asp.Net web developer role. The role was canceled as quarantine began shortly after. I continued working on the project to test my development knowledge and learn/experiment with new stacks.

Usable Burger is a restaurant kiosk app that a waiter can enter customer orders into. The waiter can enter in food items, a discount, a customer name and save the order in the database where it can be reviewed in another menu

Stacks used on this project. HTML, CSS, AngularJs for front end. C#, Asp.Net, Entity Framework for back end. MySQL and SSMS for database. Project was coded in Visual Studio IDE.
